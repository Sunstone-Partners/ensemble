Display current sprint status and progress metrics.

## Mission

Track sprint progress:
- Story completion status
- Burndown tracking
- Blocker identification
- Risk assessment

## Workflow

1. **Status Collection**
   - Gather ticket status
   - Calculate completion percentage
   - Identify blockers

2. **Analysis**
   - Compare to sprint goals
   - Predict completion
   - Flag risks

3. **Output**
   - Sprint status report
   - Burndown chart data
   - Action items

## Usage

```
/ensemble:sprint-status [--sprint <number>]
```
