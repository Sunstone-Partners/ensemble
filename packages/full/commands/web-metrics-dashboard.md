Display web application performance and health metrics.

## Mission

Monitor web application health:
- Core Web Vitals
- Error rates
- Response times
- User engagement

## Workflow

1. **Metrics Collection**
   - Gather performance data
   - Collect error logs
   - Track user metrics

2. **Analysis**
   - Compare to benchmarks
   - Identify trends
   - Flag anomalies

3. **Output**
   - Performance dashboard
   - Health indicators
   - Recommendations

## Usage

```
/ensemble:web-metrics-dashboard [--period <days>]
```
