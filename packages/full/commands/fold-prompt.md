Fold long prompts into shorter, more efficient versions while preserving intent.

## Mission

Optimize prompts for efficiency:
- Reduce token usage
- Preserve essential context
- Maintain prompt effectiveness

## Workflow

1. **Analysis**
   - Parse original prompt
   - Identify essential vs redundant content
   - Map key instructions

2. **Optimization**
   - Remove redundancy
   - Compress while preserving intent
   - Validate folded prompt

3. **Output**
   - Optimized prompt
   - Token savings report

## Usage

```
/ensemble:fold-prompt <long prompt to optimize>
```
