# Changelog

## 5.3.0

- Initial Codex CLI compatibility package
- Generates Codex agent configs, command skills, and copied Ensemble skills

- Adds version-sync automation and publish workflow improvements for npm release management
- Adds generated release notes and changelog support for Codex package releases
- Publishes a leaner runtime bundle by shipping framework skills as SKILL.md-only runtime assets
